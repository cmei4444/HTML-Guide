HTML Review Guide

Basic
  • Basic structure of an HTML Document:
    • You must have <!DOCTYPE html> at the beginning of every document.
    • Make sure to include <html>, <head>, and <body> opening and closing tags.
    • Use <link> tag to link your html document to a css stylesheet.
  • Anchor + Anchor Style: Use <a> before the link text and </a> after the link text; inside the <a> tag, after <a, put href="(your link)". Between the tags, put the content your link will display as.
    Example: <a href="http://www.google.com">Link to Google</a>
    Link to Google
  • Image: Use <img> tag, after <img, inside the tag, put src="(picture link)".
    Example: <img src="http://upload.wikimedia.org/wikipedia/commons/2/22/Turkish_Van_Cat.jpg">
  • Lists:
    • Ordered Lists: These are numbered lists. To make one, use <ol> before and </ol> after the list.
      1. This is an example of an ordered list.
      2. This is the second thing in your list.
    • Unordered Lists: These are bulleted lists. This is an example of an unordered list. To make one, use <ul> before and </ul> after the list.
    Use <li> before and </li> after each object in a list.
  • Paragraphs: Use <p> before and </p> after your paragraph.

    This is an example paragraph. This is a wonderful paragraph. Worship this paragraph. Meow meow meow meow meow meow meow meow meow meow meow meow meow meow meow meow meow meow.

Intermediate
  • Section tags: Use <section> before and </section> after your intended section.
  • Fonts: In the <style> section, under what you want the specific font to be for, put font: (font-weight)(font-size) (font-family);
    Example: The font for this page's text:
    font: normal 14px "Trebuchet MS", sans-serif
  • Image styles (including border):
    • Height/ Width: In the <img> tag, use height="(desired height)" and width="(desired width)". This can also be done in style, under img.
    • Border: In the <style> section, under img, use border: style color weight
      Different border styles:
      • Solid
      • Dotted
      • Dashed
      • Double
      • Groove
      • Ridge
      • Inset
      • Outset
    Example: border: outset mediumAquamarine 6px
  • Table
    This is a table. It is fancy. Use <table> and </table> tags before and after a table.
    This is a new row of the table. To make a row, use <tr> before and </tr> after your row. This is a table heading cell. Use <th> and </th> around the text.
    This is a table data cell. To make a table data cell, use <tr> before and </tr> after the text.
Advanced
  • Margin: In <style>, under whatever you want the margin to be for, use margin-top, margin-bottom, margin-left, and margin-right; or use margin: top right bottom left as a shortcut.
    Example: margin: 75px 50px 75px 50px;
  • Shadow:
    • Box shadow: For images, put in <style> under the image(s) you want to have a shadow. Use box-shadow: (horizontal)(vertical)(blur)(spread)(color)
      Example: box-shadow: 10px 10px 5px 3px mediumAquamarine
    • Text shadow: for text, put in <style> under whatever text you want to have a shadow. You can have multiple shadows on the same text. Use text-shadow: (horizontal)(vertical)(blur)(spread)(color)
    Example: For the heading of this page: text-shadow: 6px 3px 4px aquamarine
  • Background Image: In <style>, under whatever you want the background to be for, use background-image: url("(link to the image you want to use)");
    Example for this page: background-image: url("http://cdn.pcwallart.com/images/gradient-wallpaper-2.jpg");
